STATS 141B FINAL PROJECT

Provide best landmarks, restaurants, hotels, and nightlife for the specific place!

  • Classy hotel!
  • Beautiful landmarks!
  • Yummy food!

We decided to provide travel guide by:

  • Showing best landmarks, restaurants and hotels in CA using different graphs
  • Second,asking users to input a city
  • Third, display a basic overview of the city (through paragraphs and dataframe)
  • Lastly, analyze the reviews for different businesses in CA using natural language processing

We followed these steps by breaking this project into four parts.

  • Data Wrangling
  • Analyzing data and plotting
  • Getting basic information of each city
  • Analyzing the review content
